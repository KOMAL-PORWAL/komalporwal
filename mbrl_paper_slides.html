Key Challenges & Root Causes

üéØ Dynamics Model Limitations

  • Model learns dynamics near initial state well
  • Prediction error increases dramatically over longer horizons
  • Fails for states far from training distribution
  • MPC relies on accurate long-horizon predictions

üí• Fundamental MBRL Challenges

  • Epistemic Uncertainty: Limited data causes model errors
  • Aleatoric Uncertainty: Inherent environment stochasticity
  • Value Function Gap: MPC lacks terminal value estimate
  • Local Optima: Gradient-based planning gets stuck

üìå Why Model-Free Methods Outperform

PPO, SAC, and other model-free algorithms achieved high-quality policies using default hyperparameters on all tested benchmarks (including hardest ones like Humanoid). Model-free methods don't rely on long-horizon prediction accuracy and have better exploration mechanisms.

üí° Key Insight

Our MBRL results still outperformed prior work (Nagabandi et al.) on Hopper and Ant benchmarks, but this is not "inspiring" - basic MBRL is inherently difficult and doesn't scale well to complex control tasks without advanced techniques.

üéØ Recommended Approaches for Future Work

Hybrid Methods: Combining model-based and model-free learning (TD-MPC, MBPO, DreamerV3) show much better sample efficiency and performance on hard tasks
Terminal Value Functions: Adding learned Q-functions to MPC can mitigate horizon limitations
Latent Dynamics: Learning in learned latent representations reduces dimensionality and learning burden
Curriculum Learning: Gradually increasing task difficulty may help but requires careful tuning
